# UIDesigner
## An Interface Builder for Geometry Dash

This mod combined with [Geode DevTools](https://github.com/geode-sdk/devtools) provides an interface you can use to design and export UI menu designs, similar to Xcode's Interface Builder.

<img src="images/image1.png" alt="Showcase"/>

# WIP notice!!

As of right now, this mod requires a nightly version of geode.devtools. You will need to build that before you can build this.

## Built-in elements

- Node/Menu
- Label
- Sprite
- Row Node
- Column Node
- Markdown Text Area
- Scale9Sprite
- Sprite Button
- Scale9Sprite Button
- Text Button
